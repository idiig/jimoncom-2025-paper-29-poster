* 結論
- 隣接歌集間で語彙差が大きいペアが存在しなかった
  + 「古今→後撰」の変化が小さい傾向
  + 「拾遺→後拾遺」「金葉→詞花」の変化が大きい傾向
- 語彙差の変動のパスは文学史の説明に対応あり
- 三水準での結果が一致の傾向
  + →題材差が大きい資料で再検討すべき
